# the-field

Welcome to [**The Field**](the-field.streamlit.app), a simple app where we can track how our drafter teams/players are performing and check live odds for all events

Have an idea for a new feature? Request it [here](https://github.com/martinbog19/the-field/FEATURES.md)!

To develop locally, simply run the following command in a terminal:

```
make app
```
